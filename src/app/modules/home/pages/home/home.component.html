PlayApp

Basic App in charge of the CRUD of two entities.

  • Album
  • Song

Tecnologies

This app is served with Nginx in a ubuntu machine.

Frontend:

  • Angular CLI
    • angular/material
    • ngrx/store
    • ngx-image-cropper
    • ngx-ui-loader
    • sweetalert2
    • moment
    • lodash
    • SoundCloud Widget Api
  • Angular Material

BackEnd:

  • .Net Core
    • Entity Framework
    • AutoMapper
    • Cloudinary .Net

Database:

  • Postgresql

Cloud:

  • Cloudinary (Images)

Notes:

  • In order to add a new song you must provide a soundcloud url.
